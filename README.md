# Product Category Classification using Machine Learning

This project focuses on building a machine learning model that can automatically classify product categories based on product names and descriptions. It simulates how e-commerce platforms organize and manage large product catalogs efficiently.

## 🚀 Project Overview
The model takes raw product text as input and predicts its category using Natural Language Processing (NLP) techniques. The dataset used contains real-world product information, making the project practical and relevant.

## 🔧 Features
- Text preprocessing and cleaning  
- TF-IDF based feature extraction  
- Naive Bayes classification model  
- Achieved ~82% accuracy  
- Top-3 category predictions with confidence scores  

## ⚙️ Tech Stack
- Python  
- Pandas  
- NumPy  
- Scikit-learn  

## 📊 Model Workflow
1. Data Cleaning (lowercase, remove special characters)  
2. Text Vectorization using TF-IDF  
3. Train-Test Split  
4. Model Training using Naive Bayes  
5. Evaluation using Accuracy and Classification Report  

## 🔍 Example
Input: wireless bluetooth headphones  
Prediction: Electronics  

## 📌 Conclusion
This project demonstrates how machine learning and NLP can be applied to classify unstructured text data into meaningful categories. It provides hands-on experience in data preprocessing, feature engineering, and model building.
